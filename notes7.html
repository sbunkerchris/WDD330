Ajax

an API that allows asynchronous comunication witha server (dynamic)

The internet is composed of a web of 2 units: Clients and Servers.

Server-side- operations are made on the same enviroment that the server is in (server does processes)

Client-side- operations and processes are made in the browser of the user/viewer

API - application Programming interface. a collection of methods that allow access to a program or service outside the current enviroment.

GET - requests resources

POST - Request to perform an action/task

PUT - request to insert a resourc or update a resource

PATCH - request to make parcial updates

DELETE - request to delede a resource
