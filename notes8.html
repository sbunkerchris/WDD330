Preparation Notes

Transform

an API that allows style manipulation of an element

Translation

an API that allows movement of an element (up down left right). similar to position. Move the element without impacting the flow of the document.

Scaling

scale an element by a factor horizonally or vertically

Rotation

rotates element around a point (default: elements center)

Transitions

allows adjustment of css over time. Delay ect.

Canvas

API allowing ability to draw any shape. You can use this so you do not need to use images. can draw lines, shapes arcs, text, gradients, patterns, abilty to manipulate pixels.

SVG

Scalabel vector Graphic (using XML).

SVG vs Canvas

canvas - you cannot add to what is already there you need to re-draw everything

SVG - this is ultimately a file format rather than a set of methods that let you draw, AKA you cant manipulate a SVG image in the same way you would beablet to with Canvas.

Neither is good for static iamges. Really they are meant to be best used in the sense of interactions or annimations.
